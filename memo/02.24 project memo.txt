주류 페어 정보 조회 
1. 일반 조회 기능
사용자가 게시글을 클릭 -> 해당 게시글 번호에 해당하는 게시글 번호, 제목, 내용, 작성일, 작성자를 선택하여 화면에 띄워줌 

2. 관리자 조회 기능 
관리자가 브라우저에서 게시글 번호를 입력 ( 입력 안하면 전체 조회 ) 후 클릭 -> 게시글 번호, 제목, 사용 여부가 조회 됨 

-> 이후 비활성화 버튼을 눌러서 사용 여부를 0으로 변경 ( JPA ) -> 다음주에 JPA 배우고 처리해보기
