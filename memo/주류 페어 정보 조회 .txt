주류 페어 정보 조회 
1. 일반 조회 기능
사용자가 게시글을 클릭 -> 해당 게시글 번호에 해당하는 게시글 번호, 제목, 내용, 작성일, 작성자를 선택하여 화면에 띄워줌   -- Service 단에서 테스트 코드 까지 작성 완료 

2.특정 검색어 검색 기능 
사용자가 검색창에서 카테고리 지정(제목, 내용, 제목+내용, 작성자)  -> 키워드 입력 -> 해당되는 글 리스트()를 화면에 띄워줌 
=> 입력 받는 제목 or 내용 or 제목+내용 or 작성자 를 DTO 객체로 만들어서 Service에 전달 -> Service는 받은 DTO 객체를 Fair 객체로 변경, Mapper 호출 -> 
Mapper에서 동적 쿼리로 쿼리 작성 ->     

3. 관리자 조회 기능 
관리자가 브라우저에서 게시글 번호를 입력 ( 입력 안하면 전체 조회 ) 후 클릭 -> 게시글 번호, 제목, 사용 여부가 조회 됨
